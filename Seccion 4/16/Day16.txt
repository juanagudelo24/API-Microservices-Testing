pom.xml
----------
2 plug ins are required to execute project through pom.xml file
	1) maven-compiler-plugin
	2) maven-surefire plugin

Execute test cases though command prompt:
-----------------------------
Pre-requisite: Maven software downlaod & install

Maven
-----
	--> available as a plugin within eclispse
	--> available as software on OS
How to install Maven on windows os:
-----
1) downlaod Maven & Extract zip file
http://mirrors.estointernet.in/apache/maven/maven-3/3.6.1/binaries/apache-maven-3.6.1-bin.zip

2) Set Maven Path in Environment varaibles...

3) Verify maven installed on windows or not

Open command prompt:
	cd C:\Users\admin\eclipse-workspace\RestassuredAPITesting_Employee_Project
	mvn clean install

create run.bat file


Execute test cases using Jenkins
---------------------------------
Downlad Jenkins: http://mirrors.jenkins.io/war-stable/latest/jenkins.war

open command prompt, go to file location:

java -jar jenkins.war
